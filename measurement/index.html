How to run the scripts?

  1. Requirements: We use PySpark for big data analysis, python3 for scripting, and gnuplot for plotting scripts from data.
  2. Download the scripts in any directory, create a subdirectory called temp within the directory where the scripts reside
  3. For reproducing Table 1 and 2, run spark-submit generate-table1-table2-data.py. Match the outputs with table 1 and table 2 data.
  4. For reproducing Table 3 and generating temporary data needed to plot Figure 3, run spark-submit generate-table3-fig3-data.py. Match the outputs with table 3 data.
  5. For producing temporary data needed to plot Figure 4, run python3 generate-fig4-data.py
  6. For producing temporary data needed to plot Figure 7, run python3 generate-fig7-data.py
  7. Unzip plotting-scripts.zip in a subdirectory called plots and use gnuplot to plot the figures.
